## To Install - For Developers

```bash
# Alternatively, you can clone this repo using GitKraken
# Clone this repository
git clone https://github.com/stefnotch/youtrack-diagram
# Go into the repository
cd youtrack-diagram

# Install dependencies
npm install
# Run the app
npm start

# To build it
npm run build
```
